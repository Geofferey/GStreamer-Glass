# GStreamer Glass

GStreamer Glass is a Windows PowerShell frontend for building and running low-latency GStreamer streaming pipelines.

This repository was seeded from recovered project snapshots. Older point releases and later experimental checkpoints are preserved as linear Git history, with the active source kept at `GStreamer-Glass.ps1`.

## Contents

- `GStreamer-Glass.ps1` - current PowerShell application source.
- `gstwebrtc-api/` - packaged web player assets used by later WebRTC builds.
- `GStreamer Glass.exe.config` - application runtime config captured from the packaged build.
- `GStreamer_Glass.psccfg` - PowerShell packaging configuration.

Compiled binaries are intentionally not tracked in the seeded source history.
